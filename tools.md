### [YouTube Channel ID Finder](https://youtube-channel-id-finder-660825558664.europe-west1.run.app/)

A sleek, modern web application that extracts the Channel ID from any YouTube video URL using the YouTube Data API v3.

*   **功能**：快速從影片連結提取頻道 ID。
*   **技術**：YouTube Data API v3, Cloud Run。
*   **特色**：介面簡潔、反應迅速。
